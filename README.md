# Pipeline

## Commands

* `auth`
    * `auth -p <api-key>`  
        Set the API key
    * `auth api-key`  
        Output the current API key

* `consensus`  
    Run consensus algorithms

* `identification`  
    Run identification algorithms

* `comm`
    * `comm -m online`  
        Connect to the Database API
    * `comm -m offline`  
        Store data on disk
    * `comm sync`  
        Synchronise local data with the online database
